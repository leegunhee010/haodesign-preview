안내

이 관리자는 시안용으로 이 브라우저에만 저장됩니다(localStorage). 히어로 · 포트폴리오 · 칼럼 · 연락처를 수정하면 같은 브라우저에서 보는 사이트에 즉시 반영됩니다. 실서비스에서는 서버(DB · 인증) 연동으로 교체가 필요합니다.

히어로 배너

메인 상단 슬라이드 3개의 문구와 이미지를 관리합니다. 헤드라인 · 설명의 **단어** 는 오렌지 강조로 표시됩니다.

카피 수정

✏️ 사이트에서 바로 수정 — 아래 버튼으로 페이지를 열면 점선 표시된 글을 화면에서 직접 클릭해 고칠 수 있습니다.

메인 ↗ 회사소개 ↗ 지원사업 ↗ 포트폴리오 ↗ 칼럼 ↗ 견적문의 ↗

서비스 페이지 — 카테고리별로 열어 헤드카피 · 설명 · 소개글을 직접 수정하세요.

카탈로그·브로슈어 ↗ 팜플렛·리플렛 ↗ 포스터·전단지 ↗ CI·로고디자인 ↗ 패키지·라벨 ↗ 촬영서비스 ↗ 다국어디자인 ↗

또는 아래에서 직접 수정해도 됩니다. **단어** 는 강조 표시, 줄바꿈은 그대로 반영됩니다.

서비스 페이지

서비스 7개 카테고리 페이지의 카피를 관리합니다. 헤드카피의 **단어** 는 강조 표시, 줄바꿈은 그대로 반영됩니다. (헤더 메뉴 이름은 코드에서 변경)

포트폴리오

체크한 작품이 먼저 노출되고, 부족하면 나머지 포트폴리오로 자동 채워집니다.

흐르는 띠 (롤링바)

메인 페이지 상단에서 좌우로 흐르는 띠에 들어갈 사진입니다. 슬롯마다 사진을 직접 올려서 바꾸세요.
※ 아래 제작 사례 모자이크는 이제 포트폴리오에서 '메인 노출' 체크한 작품으로 자동 구성됩니다. (비워두면 흐르는 띠도 그 작품들로 채워집니다.)

칼럼

글을 누르면 수정 화면이 열립니다. 목록은 날짜순으로 자동 정렬됩니다.

글쓰기

고객 후기

메인 페이지 "다시 찾아주시는 이유" 섹션의 후기 카드를 관리합니다. 위에서부터 순서대로 두 줄에 나뉘어 흐릅니다.

페이지별 사진

각 페이지에 들어가는 사진을 자리별로 직접 바꿉니다. 칸을 클릭해 사진을 올리면 그 자리에 고정되고, × 를 누르면 다시 포트폴리오 자동 선택으로 돌아갑니다.

※ 서비스 페이지 작업 사례는 이제 포트폴리오에서 '서비스 노출'을 체크한 같은 카테고리 작품으로 자동 구성됩니다. 빈 칸은 포트폴리오에서 자동으로 채워집니다.
